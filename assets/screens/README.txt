Скриншоты тренажёра для блока «Знакомство с продуктом» на salespro.html.
Имена файлов должны совпадать (иначе на сайте покажется заглушка):

  fight.png — экран боя (диалог с ИИ-клиентом + коучинг в реальном времени)
  ball.png  — итог боя (общий балл, XP, новое звание, оценка по навыкам)

Картинки показываются целиком (без обрезки), поэтому подойдёт любой размер.
Чтобы заменить — просто положите новый файл с тем же именем.
Если добавите ещё экраны (кабинет, дашборд РОПа) — скажите, добавлю вкладки.
