# Ассеты сайта TTT Marketing

Все изображения подключаются из `index.html` по имени файла.

## Что где

| Файлы | Назначение |
|---|---|
| `newlogo.png` | Исходный логотип (квадрат, прозрачный фон). |
| `newlogo-mark.png` | Обрезанный логотип-марка — используется везде: фавикон, шапка, орбита, прелоадер, иконки. |
| `system.jpg` | Фон секции «СИСТЕМА». |
| `client-*.jpg` | Фото кейсов в слайдере «200+ проектов» (оптимизированные, ≤1100px). |
| `AltynOrda.webp`, `AppleResort.webp`, `Biotmin.webp`, `Miracle.webp`, `CLUB100.jpg`, `Corsa.jpg`, `GDAGroup.jpg`, `Qulpynai.jpg`, `Sman.jpg`, `Tagam.jpg` | Остальные фото кейсов (подключены напрямую). |
| `team-*.jpg` | Фото сотрудников в блоке «Наша команда» (11 шт.). |
| `og-cover.svg` | Исходник соц-обложки (Open Graph). |

## Соглашение по именам

Новые фото — латиницей, без пробелов. Кейсы: `client-<бренд>.jpg`, команда: `team-<имя>.jpg`.
Тяжёлые исходники (PNG > 1 МБ) перед добавлением сжимать в JPG/WebP (ширина ≤ ~1100px).
