## Про og-cover.svg

`og-cover.svg` — векторная соц-обложка в фирменном стиле (тёмный фон, триколор, слоган, ключевые цифры). Чтобы соцсети показывали превью, её надо превратить в `og-cover.png` (1200×630). Варианты:
- открыть SVG в браузере → скриншот 1200×630;
- любой онлайн-конвертер SVG→PNG;
- когда поставим Node.js — конвертну программно (`sharp`).
