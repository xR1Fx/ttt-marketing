# TTT Marketing — лендинг

Одностраничный сайт маркетингового агентства **TTT Marketing** (Алматы).
Тёмная премиум-тема, фирменный триколор Google (синий/жёлтый/красный), шрифт Inter.

## Стек

- **Чистый HTML/CSS/JS в одном файле** `index.html` (без сборки, без npm).
- **GSAP + ScrollTrigger** (CDN) — скролл-анимации, пиннинг.
- **Lenis** (CDN) — инерционный smooth-scroll.
- Фон — анимированная canvas-«сеть»; фоны разделителей — анимированные SVG в `assets/`.

## Как запустить

Node не нужен — сайт статический.

1. Открой папку в **VS Code**.
2. Установи расширение **Live Server** и нажми **Go Live** (или просто открой `index.html` в браузере).
3. Авто-перезагрузка при сохранении.
