# TTT Marketing — лендинг

Одностраничный сайт маркетингового агентства **TTT Marketing** (Алматы).
Тёмная премиум-тема, фирменный триколор (синий/жёлтый/красный), шрифт Inter.

## Стек

- **Чистый HTML/CSS/JS в одном файле** `index.html` (без сборки, без npm).
- **GSAP + ScrollTrigger** (CDN) — скролл-анимации и появления.
- **Lenis** (CDN) — инерционный smooth-scroll.
- Фон — анимированная canvas-«сеть» (частицы); логотип — `assets/newlogo-mark.png`.

## Как запустить

Node не нужен — сайт статический.

1. Открыть папку в **VS Code**.
2. Установить расширение **Live Server** → **Go Live** (или просто открыть `index.html` в браузере).
3. Авто-перезагрузка при сохранении.

## Формы

Backend нет — обе формы (заявка и обратная связь) при отправке **открывают WhatsApp** с заполненными
полями (Имя / Телефон / Сообщение). Чтобы заявки приходили на почту с вложениями, нужно подключить
сервис форм (Formspree / Web3Forms) — поменять `action`/обработчик отправки.

## Адаптив и доступность

Полная мобильная вёрстка (свайп-ленты для кейсов и команды), `prefers-reduced-motion`,
семантические теги, SEO/OG-мета.
